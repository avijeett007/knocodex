# Knocodex Assets

This directory contains the assets for the Knocodex website.

## Logo

The Knocodex logo represents the combination of AI and coding:

- The code brackets (`<>`) symbolize programming and development
- The circuit pattern in the center represents AI and machine learning
- The blue color scheme is associated with technology and intelligence

## Favicon

The favicon is a simplified version of the logo, designed to be recognizable at small sizes.

## Usage Guidelines

When using the Knocodex logo:

1. Do not alter the colors or proportions
2. Maintain adequate spacing around the logo
3. Do not use the logo in a way that suggests endorsement of unrelated products or services
4. When using on dark backgrounds, use the standard version; on light backgrounds, use the inverse version

## Brand Colors

- Primary Blue: #0073ff
- Dark Background: #0c0c0c
- Terminal Green: #00ff00
- Success Green: #33ff33
- Error Red: #ff3333

## Typography

- Headings: Inter, system-ui, sans-serif
- Body: Inter, system-ui, sans-serif
- Code: Menlo, Monaco, Consolas, Liberation Mono, Courier New, monospace
