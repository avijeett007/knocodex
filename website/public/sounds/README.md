# Sound Effects

This directory contains sound effects used in the Knocodex website.

## Files

- `key-press.mp3`: A subtle keyboard key press sound
- `command-success.mp3`: Sound played when a command completes successfully
- `command-error.mp3`: Sound played when a command results in an error

## Usage

These sound effects are used in the Terminal component to provide a more immersive experience.
